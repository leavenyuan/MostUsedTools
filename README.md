#### python requirements生成与使用
    方法1：pip freeze > requirement.txt (其中freeze参数是指输出当前环境中安装的依赖包,适用于anaconda管理环境的项目)
    方法2：pip install pipreqs
           pipreqs . --encoding=utf8 --force  # 在当前目录生成
    使用：pip install -r requirements.txt
